## What Gets Preserved

At 10x compression, AXL preserves:
- All named entities (patients, agents, countries, organizations)
- All numeric values (CA-125 levels, troop counts, BTC prices, p-values)
- All confidence levels (per-agent, per-claim)
- All causal relationships (evidence chains, dependency graphs)
- All temporal scopes (NOW, 1H, 1D, 1W)
- All cognitive operations (who observed, inferred, contradicted, predicted)

## What Gets Lost

- Rhetorical style and prose cadence
- Hedging language ("it appears that", "one might argue")
- Redundant restatements
- Filler phrases
- Paragraph structure

## Fidelity Measurement

The decompression fidelity score measures round-trip quality:

```
F = 100 x (0.35*Ssem + 0.20*Snum + 0.15*Srel + 0.10*Stemp + 0.10*Sonto + 0.10*Sconf)
```

BG-007 achieved F = 97.6 — doctrine-grade round-tripping where all semantic content survives compression and decompression.
