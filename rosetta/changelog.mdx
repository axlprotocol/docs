## v3 - March 28, 2026

75 lines. Complete rewrite as BNF grammar kernel.

- Replaced 445-line documentation-style spec with pure BNF grammar
- PKT format: `ID|OP.CC|SUBJ|ARG1|ARG2|TEMP [META]`
- Loss contracts required for all bundles (mode, keep, loss, fidelity)
- 3-step decompression pipeline with fidelity formula
- JSON lowering via `application/vnd.axl+json` with JCS (RFC 8785) canonical form
- Genesis tracking: agents report first parse to bridge.axlprotocol.org
- Full backward compatibility with all v2.x packets
